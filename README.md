# 💈 Grooming Hub

A web-based grooming services booking and management system built as my 
Final Year Project for the Diploma in Information Communication Technology 
at Kabete National Polytechnic.

---

## 📌 About The Project

Grooming Hub is a full-stack web application designed to help grooming 
businesses manage their services, appointments, and customers efficiently. 
It replaces manual booking systems with a clean, easy-to-use digital platform.

---

## ✨ Features

- 🗓️ Online appointment booking system
- 👤 Customer registration and login
- 💼 Service listing and management
- 🔐 Admin dashboard for managing bookings
- 📱 Responsive design — works on mobile and desktop
- 🗄️ MySQL database for data storage and retrieval

---

## 🛠️ Built With

| Technology | Purpose |
|---|---|
| HTML5 | Structure and layout |
| CSS3 | Styling and responsive design |
| PHP | Backend logic and server-side processing |
| MySQL | Database management |
| JavaScript | Frontend interactivity |

---

## 🚀 How To Run Locally

1. Install [XAMPP](https://www.apachefriends.org/) or any local PHP server
2. Clone this repository into your `htdocs` folder
3. Import the database file into phpMyAdmin
4. Open your browser and go to `http://localhost/grooming-hub`
